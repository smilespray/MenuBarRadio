# MenuBarRadio

The simplest menu bar radio app for OS X

##One Feature
- Internet radio while you work.

##Many Unfeatures
- No volume control
- No EQ
- No Recording
- No bloated UI
- No artwork retrieval
- No lyrics retrieval
- No music store integration
- No social media integration
- No registration
- No spam
